# 02_lab020106_R: Грамотное программирование в R

## Описание проекта
Данный проект содержит реализацию грамотного программирования на языке R с использованием технологии R Markdown. В рамках лабораторной работы создан динамический документ, демонстрирующий основные принципы объединения кода, результатов вычислений и текстовых описаний в едином документе.

## Задания

### Задание 1: Создание динамического документа с R Markdown
- Создан R Markdown документ с YAML-заголовком и метаданными
- Реализованы различные типы вставок кода: чанки и inline-выражения
- Использованы различные параметры чанков (echo, include, results)
- Включены математические формулы и форматирование текста
- Применена каскадная таблица стилей для кастомизации внешнего вида
- Документ сконвертирован в HTML формат

## Требования
- R версии 4.0.0 или выше
- RStudio (рекомендуется)
- Пакет `rmarkdown`
- Пакет `knitr`

## Примеры использования
*Создание динамического контента:*
```r
    # Inline-выражения в тексте
    Встроенный набор данных `cars` содержит `r nrow(cars)` наблюдений.

    # Чанки с кодом
    ```{r calculations, include=FALSE}
    avg_speed <- mean(cars$speed)
    ```
```
Средняя скорость: `r round(avg_speed, 1)` миль/ч


*Кастомизация через CSS:*
```css
h1 {
    color: #2c3e50;

}
```
Принципы грамотного программирования, использованные в проекте
1. **Динамические документы - объединение кода, результатов и текста**
2. **Воспроизводимость - возможность повторного расчета с обновленными данными**
3. **Интеграция кода и текста - seamless-взаимодействие R и Markdown**
4. **Профессиональное оформление - использование CSS для кастомизации**
5. **Гибкость форматов - поддержка HTML, PDF, Word выходных форматов**

Ключевые элементы R Markdown
- **YAML-заголовок - метаданные документа и настройки вывода**
- **Чанки (chunks) - блоки исполняемого кода с различными параметрами**
- **Inline-код - встроенные R-выражения непосредственно в тексте**
- **Markdown-разметка - форматирование текста (заголовки, списки, таблицы)**
- **CSS-стили - кастомизация внешнего вида HTML-документов**
