# 02_lab020105_R: Функциональное программирование в R

## Описание проекта
Данный проект содержит реализацию функционального программирования на языке R с использованием пакета purrr. В рамках лабораторной работы реализованы два задания, демонстрирующие основные принципы работы с итерациями и конвейерами в функциональном стиле.

## Задания

### Задание 1: Создание именованного списка фильмов
- Использованы данные пакета `repurrrsive` со списком `sw_films`
- Создан именованный список с названиями фильмов в качестве имен элементов
- Реализовано два подхода: с промежуточным сохранением и с использованием конвейера
- Продемонстрирован доступ к элементам списка по имени

### Задание 2: Исследование функций семейства map*
- Изучены и продемонстрированы различные вариации функций семейства `map`
- Созданы тестовые данные для демонстрации работы функций
- Показаны особенности каждой функции и типы возвращаемых значений
- Включены примеры работы с табличными данными

## Требования
- R версии 4.0.0 или выше
- Пакет `purrr`
- Пакет `repurrrsive` (для задания 1)
- Пакет `datasets` (для задания 2)

## Принципы функционального программирования, использованные в проекте
1. **Итерации - применение функций к коллекциям данных**
2. **Конвейеры - последовательная обработка данных через оператор %>%**
3. **Чистые функции - функции без побочных эффектов**
4. **Функции высшего порядка - функции, принимающие другие функции как аргументы**
5. **Неизменяемость данных - создание новых структур вместо изменения существующих**

## Функции семейства map, использованные в проекте
`map()` - базовая функция для применения функции к каждому элементу
`map_if()` - условное применение функции
`map_at()` - применение функции к конкретным элементам
`map_lgl()` - возврат логического вектора
`map_chr()` - возврат символьного вектора
`map_int()` - возврат целочисленного вектора
`map_dbl()` - возврат числового вектора
`map_dfr()` - объединение в data frame по строкам
`map_dfc()` - объединение в data frame по столбцам
`walk()` - выполнение функций для побочных эффектов
