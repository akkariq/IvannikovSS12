# 02_lab020104_R: Векторное программирование в R

## Описание проекта
Данный проект содержит реализацию векторного программирования на языке R. В рамках лабораторной работы реализованы два задания, демонстрирующие основные принципы работы с векторами и датафреймами в R.

## Задания

### Задание 1: Предобработка данных
- Создан вектор `my_vector` с набором числовых данных
- Вычислены среднее значение и стандартное отклонение
- Отфильтрованы наблюдения, отклоняющиеся от среднего меньше чем на одно стандартное отклонение
- Результат сохранен в переменную `my_vector2`

### Задание 2: Функция для поиска отрицательных значений
- Реализована функция `get_negative_values` для работы с датафреймами
- Функция проверяет наличие отрицательных значений в каждом столбце
- Возвращает список или матрицу с отрицательными значениями в зависимости от равномерности их распределения
- Обрабатывает пропущенные значения (NA)

*Задание 2: Работа с датафреймом*
```r
# Создание тестового датафрейма
df <- as.data.frame(list(V1 = c(NA, -0.5, -0.7, -8), 
                        V2 = c(-0.3, NA, -2, -1.2),
                        V3 = c(1, 2, 3, NA)))

# Поиск отрицательных значений
result <- get_negative_values(df)
print(result)
```

## Принципы векторного программирования, использованные в проекте
1. **Векторизация операций - одновременная обработка всех элементов вектора**
2. **Работа с датафреймами - обработка табличных структур данных**
3. **Обработка пропущенных значений - корректная работа с NA**
4. **Условные операции - фильтрация данных по заданным критериям**
5. **Гибкость возвращаемых значений - адаптация типа результата в зависимости от данных**
