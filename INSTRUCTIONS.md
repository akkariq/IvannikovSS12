# Полные Нинструкции по объединению Лабораторных Работ

## Шаг 1: Клонирование Репозитория

Открыте терминал и выполните:

```bash
git clone https://github.com/akkariq/IvannikovSS12.git
cd IvannikovSS12
```

## Шаг 2: Выполнение Настройки

После клонирования, запустите скрипт автоассемблера:

### На Linux/Mac:
```bash
bash setup_labs.sh
```

### На Windows (PowerShell):
```powershell
# Сначала убедитесь, что у вас установлен Python и Git
bash setup_labs.sh
```

## Что делает этот скрипт:

1. Подключает все репозитории источников
2. Копирует все папки (структура: lab1/, lab2/, lab4/, lab6/, lab7/, lab_oop1/, lab_oop2/, lab_oop3/)
3. Заменяет имя студента ("Лебский Артём Александрович" -> "Иванников Сергей Сергеевич") во всех текстовых файлах
4. Наддает все папки для коммита
5. Выкакивает исправленные работы на GitHub

## Энти папки будут сценаритесь:

- **lab1/** - Лабораторная работа Организация программирования (Python)
- **lab2/** - Лабораторная работа R программирования
- **lab4/** - ООП на Python
- **lab6/** - Функциональное программирование
- **lab7/** - Программирование на Go
- **lab_oop1-3/** - Дополнительные объектно-ориентированные работы (R)
